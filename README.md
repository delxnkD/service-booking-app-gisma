# Service Booking Web Application  

The **Service Booking Web Application** is a full-stack platform developed using **Node.js, Express, and MongoDB**. It enables users to register, log in, browse available services, and book appointments seamlessly through a user-friendly interface.  

The system implements RESTful APIs to handle client-server communication and supports dynamic front-end rendering for an interactive user experience. Role-based access control is integrated to distinguish between regular users and administrators.  

Users can:
- Create and manage accounts  
- View available services  
- Book and track bookings 
- Optionally - Cancel pending bookings 

Administrators can:
- Add, update, or delete services  
- Manage user accounts  
- View and update booking statuses  

The platform improves efficiency in service management.
